# OUTBRAIN 프론트엔드

> **내 기억 구조를 직접 들여다보는 앱**
> 목록이 아닌 그래프로 기록을 탐색합니다.

## UX 개요

**검색 = 기억 그래프를 여는 행위**

- 리스트 중심 UI 없음 — 검색 결과는 바로 그래프로 표시
- 사건(Event)이 항상 중심 — 노드 = 사건
- 탐색은 선형이 아니라 공간적 — 스크롤이 아니라 확대·축소·이동
- 맥락 유지 — 새 검색·질문을 해도 기존 그래프 위에 누적

## 유저 플로우

1. **검색** — 상단 검색창에 키워드 또는 자연어 입력
2. **그래프 열림** — 중앙 노드 + 주변 노드(동일 인물/감정/유사 사건)가 radial 레이아웃으로 표시
3. **노드 클릭** — 요약(날짜, 인물, 감정) + [원문 보기] 버튼
4. **확장** — 동일 인물 / 동일 감정 / 유사 사건으로 그래프 확장
5. **질문** (선택) — 그래프 위에 관련 노드·간선 강조 누적
6. **타임라인** (보조) — 시점별 기록 탐색

## 실행

```bash
npm install
npm run dev
```
